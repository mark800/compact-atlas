# compact-atlas

convert apache atlas to a spring-boot project, easy to use.  

====================
1. enter the home folder  
2. mvn package  
3. java -jar target/compactatlas-0.0.1-SNAPSHOT-exec.jar  

open: http://localhost:9090/index.html  
login: admin/admin
